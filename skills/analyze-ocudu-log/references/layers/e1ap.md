# E1AP layer

## What this layer reveals

The E1 Application Protocol is the CU-CP–CU-UP interface. E1AP logs show bearer setup and release for the user-plane split. Useful when diagnosing data-plane failures where control-plane setup appears to succeed (UE is RRC Connected) but no user data flows.

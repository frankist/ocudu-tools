# RRC layer

## What this layer reveals

Radio Resource Control manages UE connection setup, reconfiguration, and release, as well as system information broadcast (SIBs). RRC logs are the right place to look when a UE completes RACH but never reaches a connected state, or when connections drop unexpectedly.
